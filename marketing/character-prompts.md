
Visual Style Guide: Corporate Memphis Edition
Style Keywords: Corporate Memphis, Flat illustration, geometric shapes, vibrant color palette, exaggerated proportions, thick flowing limbs, minimal facial features, grainy texture, clean vector style, joyful but professional.
1. EOFP — 玻璃大炮 / The Glass Cannon
视觉描述： 一个拥有修长、半透明蓝色肢体的人物。躯干是一个充满流动二进制代码的玻璃瓶。手臂极长，正以夸张的动作敲击漂浮的几何按键。
孟菲斯元素： 身体表面有细密的白色矢量裂纹，周围漂浮着粉色和黄色的方形工作流碎片。
2. EOFH — 人脉桥梁 / The Human Bridge
视觉描述： 人物叉腰站立，双臂化作无数条细长的曲线，向画面四周延伸，连接着代表不同人的彩色圆点。
孟菲斯元素： 胸口有一个巨大的明黄色圆形，代表核心。背景是交织的线条网格，地面由不规则的拼图色块组成。
3. EORP — 终审印章 / The Final Stamp
视觉描述： 人物拥有巨大的、圆柱状的手臂，末端抓着一个比头还大的红色几何印章。另一只手掌呈巨大的平面，做出阻拦姿势。
孟菲斯元素： 身后是层叠的巨大矩形（代表文件），脚下是一根贯穿画面的粗壮红线。
4. ESFP — 品味定义者 / The Taste Maker
视觉描述： 人物站在一堆灰色、重复的三角形中，唯独指向其中一个彩虹色的圆形。人物的头是一个发光的棱镜球。
孟菲斯元素： 极长且弯曲的手指，极简的五官中只有一只眼睛是闪烁的星形。
5. TOFP — 赤手行者 / The Bare Hand
视觉描述： 人物拥有异常粗大的明橙色双手，正握着一段弯曲的、蓝色的矢量管道。身体其他部分较小，显得低调。
孟菲斯元素： 管道周围有齿轮状的几何图形，背景是实心的波纹线条，强调“物理接触”。
6. EORH — 执照之墙 / The License Wall
视觉描述： 人物身穿深蓝色的巨大梯形西装，被一圈发光的半透明方块（证书）环绕成一个圆阵。
孟菲斯元素： 细长的双腿，戴着一个夸张的大圆形黑框眼镜，光栅线条在保护罩表面流动。
7. ESFH — 活体品牌 / The Living Brand
视觉描述： 一个跳跃的人物姿态，身体散发出炫目的波普圆点。周围漂浮着各种个人标识（如星星、爱心、对话框）。
孟菲斯元素： 极其扭曲且动感的肢体线条，背景是一个巨大的黄色圆形，像光环一样衬托出人物剪影。
8. ESRP — 高压炼金师 / The Pressure Alchemist
视觉描述： 人物两只手分别托举着一个巨大的实心圆球和一个脆弱的三角形，身体像一个平衡的支点。
孟菲斯元素： 沉稳的深紫色调，背景有倾斜的线条代表重力感，人物双腿像柱子一样稳固。
9. TOFH — 签名手艺人 / The Signature Touch
视觉描述： 画面中心是一双灵活交织的长手，指尖带起一串优美的波浪线，抚摸着一个抽象的人物轮廓。
孟菲斯元素： 温暖的粉红色和浅橘色调。背景充满了手绘感的螺旋曲线，代表“个人签名”。
10. TORP — 不颤之手 / The Steady Hand
视觉描述： 全景灰调中，只有人物的一只亮绿色手臂是绝对垂直平稳的，指尖顶着一颗极小的珍珠。
孟菲斯元素： 画面其他部分的线条都是波浪状或模糊的，唯独人物由极其锋利的直线和完美的圆形构成。
11. TSFP — 灵魂匠人 / The Soul Craftsman
视觉描述： 人物弯腰捧着一个形状不规则、边缘粗糙但闪烁金光的几何体。人物身上有各种彩色的泼墨色块。
孟菲斯元素： 不对称的肩膀设计，强调“手工感”的不规则多边形背景，色调充满泥土气息。
12. ESRH — 神谕者 / The Oracle
视觉描述： 人物坐在一座由无数层叠书本构成的阶梯顶端，头顶悬浮着由数字和年份组成的巨大光环。
孟菲斯元素： 极长且低垂的手臂，深沉的靛蓝色调，背景是无数向上延伸的细长三角箭头。
13. TORH — 疗愈之手 / The Healing Hand
视觉描述： 两个人物的互动，主体人物的手掌异常宽大，散发出柔和的脉冲波纹，覆盖在另一个缩小的人物身上。
孟菲斯元素： 温暖的薄荷绿和珊瑚色，流线型的肢体连接，画面充满圆润的弧线。
14. TSFH — 不可替代者 / The Irreplaceable
视觉描述： 人物的身体像一道流动的彩虹，没有固定的形状，在画面中舞动。周围的几何背景随其动作而扭曲。
孟菲斯元素： 极致的色彩对比，无面部特征，全靠极具张力的肢体语言和负空间来表达存在感。
15. TSRP — 终极裁决者 / The Last Call
视觉描述： 画面左侧是混乱的黑色乱麻线条，右侧是整齐的亮蓝色平行线。人物站在中间，手像剪刀一样切断混乱。
孟菲斯元素： 强烈的几何对立，人物穿着带有硬朗线条的背心，表情由两个坚定的横杠（眼睛）组成。
16. TSRH — 铁壁堡垒 / The Iron Fortress
视觉描述： 人物正襟危坐，身体被四个巨大的、色彩各异的同心圆环（护盾）严密包裹。
孟菲斯元素： 极致的对称美。最外层护盾有类似城墙的锯齿边缘。金黄与深蓝的配色，散发出不可撼动的权威感。
