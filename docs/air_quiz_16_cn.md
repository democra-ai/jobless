# AIR 16题精简版问卷

> 从60题完整版中精选，每维度4题（覆盖4个不同facet），总计8正向+8反向
>
> 日期: 2026-03-25

---

## 选题原则

1. **Facet覆盖最大化**：每维度5个facet中选4个，确保内容效度
2. **正反向完美平衡**：每维度2F+2R，总计8F/8R = 50/50
3. **高区分度优先**：选行为具体、选项梯度清晰、所有行业通用的题目
4. **避免维度交叉**：排除可能同时测量两个维度的题目
5. **按维度分组**：同一维度的4题连续呈现，按D1→D2→D3→D4顺序，维度内先易后难

---

## 符号说明

- **D1** = 可学习性 (E/T)
- **D2** = 评判客观性 (O/S)
- **D3** = 容错性 (F/R)
- **D4** = 人格依赖性 (P/H)
- **正向**: 选项1 = 最不利于AI替代 → 选项5 = 最利于AI替代（原始分直接使用）
- **反向**: 选项1 = 最利于AI替代 → 选项5 = 最不利于AI替代（计分时 6 - score）
