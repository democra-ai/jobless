# AIR 16問簡易版アンケート

> 60問フルバージョンから厳選、各次元4問（4つの異なるファセットをカバー）、合計8正方向+8逆方向
>
> 日付: 2026-03-25

---

## 選定基準

1. **ファセットカバー率の最大化**：各次元5ファセット中4つを選定し、内容妥当性を確保
2. **正逆方向の完全バランス**：各次元2F+2R、合計8F/8R = 50/50
3. **高弁別力を優先**：行動が具体的、選択肢の段階が明確、全業種に適用可能な問題を選定
4. **次元間の交差を回避**：2つの次元を同時に測定しうる問題を除外
5. **次元ごとにグループ化**：同一次元の4問を連続提示、D1→D2→D3→D4の順、次元内はやさしい順

---

## 凡例

- **D1** = 学習可能性 (E/T)
- **D2** = 評価の客観性 (O/S)
- **D3** = リスク許容度 (F/R)
- **D4** = 人格依存度 (P/H)
- **正方向**: 選択肢1 = AI代替されにくい → 選択肢5 = AI代替されやすい（素点をそのまま使用）
- **逆方向**: 選択肢1 = AI代替されやすい → 選択肢5 = AI代替されにくい（スコア計算時 6 - score）
